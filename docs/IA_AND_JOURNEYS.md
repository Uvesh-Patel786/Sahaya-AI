# Information Architecture & User Journeys

## Navigation Map

```
Sahayak AI
├── Landing
├── Auth (Login / Register)
└── App Shell
    ├── Dashboard (Digital Twin overview + alerts)
    ├── Chat Assistant (+ Voice)
    ├── Schemes
    ├── Opportunities
    ├── Life Events
    ├── Roadmaps
    ├── Documents (Vault + Analyzer + Letter Explainer)
    ├── Civic Reports
    ├── Scam Detector
    ├── Deadlines
    └── Settings (profile, language, privacy)
```

## Key Journeys

### J1 — Find a scheme
Landing → Register → Complete Digital Twin → Schemes → View match + confidence → Apply via official link.

### J2 — Understand a letter
Documents → Upload letter → AI Explainer → Actions + deadlines → Add to Deadline Guardian.

### J3 — Report a pothole
Civic → Capture photo + pin location → AI classification → Confirm complaint → Track status.

### J4 — Verify scam SMS
Scam Detector → Paste message → Risk label + reasons → Safety tips.

### J5 — Marriage life event
Life Events → Select marriage → Checklist (docs, name change, benefits) → Link to Roadmap/Vault.
